Color is an extremely important part of any web experience. Calcite Web provides a branded palette including neutrals (grayscale) and a variety of other spot colors.

Read on for some basic guidelines and ideas for using color in your designs.

1. When in doubt, use neutral colors

A majority of your UI should use neutral colors. For this reason, Calcite Web's largest selection of swatches is the neutral set (shown below).

## 3. Use color to encourage action Color is a very good way to emphasize actionable elements within your UI. Colorful elements will naturally be higher on the visual hierarchy than neutral elements. Generally, it's best to stick with the primary color (blue) for applications like this. A good example of how color can create a hierarchy of actions can be found in buttons. With one color (blue), we can create multiple levels of importance:
btn btn-clear btn-transparent
## 4. Use sufficient contrast for text When choosing colors for text, be sure to select two colors with a sufficient contrast ratio. This makes it easier for users to read and use your site.
