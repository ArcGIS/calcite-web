What have we got?

Plug & Play

The Plug & Play Examples work right out of the box with Calcite Web. These are most of the basic, root page structures and objects that we use to build most of out websites. ANy of these examples with always work with the core Calcite Web framework.

Marketing

The marketing exention contains a set of styles and patterns used to create high-impact, engaging and dyanamic editorial style pages for hooking users and driving them deeper into a given workflow. This extension contains things like carousels, image switchers, and carousels

E-Commerce

The e-commerce extionsion supports any moment where a user needs to make a decision and complete a purchase within a website. It includes pricing tables, fancy forms, and other stuff maybe.

Dashboards

Many applications need dashboards to track user activity and content. The Dashboard extension supports activity streams, statistics, and other stuff I'm sure.
