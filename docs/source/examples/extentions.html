Overview Extentions

How Do Extentions Work?

Extensions are CSS files that sit on top of calcite web, providing additional styles for project-specific design patterns that are not ubiquitus enough to include in the primary famework for everyone everywhere to consume.

With the proper extensions in place on your project, just copy and paste the example code provided with the extention, and you're in business.
