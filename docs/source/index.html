A Design & Development Framework.

{% markdown %} Calcite Web is a branded CSS & JavaScript framework, web style guide, and visual language design catalog for Esri. You can use Calcite Web to quickly build on-brand, lightweight, and accessible websites. The robust codebase is well supported, steadily maintained, and always being improved. Calcite Web contains everything you need to quickly build excellent web sites out of the box, with no configuration or set up. A collection of [extensions](#) for the framework provide sets of patterns that can be used for specific use cases, including [marketing pages](#), [e-commerce](#), and application [dashboards](#). The sass-based system allows for use a dependency for creating branded themes for other front end frameworks, custom application-specific extensions, and other developer-friendly uses. {% endmarkdown %}
