Common Page Layouts

Aside / Main

This structure is commonly used for any big block of content that needs a secondary navigation structure. Documentation uses this often, as does a search results page, and a blog page. Can be reversed to be Main / Aside.

View the page sample.

<div class="grid-container">
  <aside class="column-6">
    <!-- Aside Content -->
    <!-- Search Filters, Side Navs, etc -->
  </aside>

  <main class="column-17 pre-1">
    <!-- Main Content -->
    <!-- Article, Search Results, other Content -->
  </main>
</div>

Three Up

When presenting a set of facets to illustrate a concept, the three up block is useful

View the page sample.

<div class="grid-container">
  <div class="column-8">
    <!-- Panel, Card, or Illustration -->
  <div class="column-8">
    <!-- Panel, Card, or Illustration -->
  </div>
  <div class="column-8">
    <!-- Panel, Card, or Illustration -->
  </div>
</div>

Content / Illustration

When the aside needs more room, either to be a set of code sample, an image, video, or other information that needs more space and is not really navigation.

View the page sample.

<div class="grid-container">
  <div class="column-12">
    <!-- Main Content -->
    <!-- Header and paragraph -->
  </div>
  <div class="column-11 pre-1">
    <!-- Illustration -->
    <!-- Video, Image, or Code Sample -->
  </div>
</div>
